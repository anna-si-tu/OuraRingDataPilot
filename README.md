This repo is for exploratory analysis on the Oura ring. 

*Terms of Service*
This application is provided for research and educational purposes only.
It is used to analyze Oura Ring data from users who explicitly authorize access.
The application is provided “as is” with no guarantees.

This application is used for academic research and personal data analysis.
It only accesses Oura Ring data for authorized users.
No data is shared with third parties.
Data is stored locally and used solely for research purposes.
Users may revoke access at any time through their Oura account.
